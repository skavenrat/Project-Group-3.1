Обзор метода System.Math.Sin() из платформы .NET Microsoft.
Рассматривается данная функция на примере языка C#.
Авторы: Фёдор Голиков, Данил Григорьев, Ярослав Лемешко, Александр Мясников, Анастасия Меркулова, Дмитрий Ткачёв, Марина Яваева.

Определение метода:
public static double Sin(double a);

Данная функция принимает единственный параметр - double a. Это вещественное число типа System.Double, принимаемое нами за радианы.
Передавая параметру вещественное число, на выходе, по результату метода, мы получим вещественное число типа double.

Примечание:
Так как параметр a (угол), должен быть в радианах, чтобы перевести градусы в радианы следует умножить их на Math.PI/180.
При передаче параметру a значений Double.NaN (неопределённость), Double.NegativeInfinity (минус бесконечность) или Double.PositiveInfinity (плюс бесконечность) функция System.Math.Sin() возвращает значение Double.NaN (неопределённость).

Примеры использования кода на языке C#:
class Program
{
    static void Main()
    {
        System.Double x = System.Double.Parse(System.Console.ReadLine()) * System.Math.PI / 180; //введение градусов
        System.Console.WriteLine(System.Math.Sin(x)); //вывод результата метода
        System.Console.ReadKey();
    }
}

Данная программа возвращает значение синуса, основываясь на введённых пользователем градусах.
